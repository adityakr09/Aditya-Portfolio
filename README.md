# Aditya Kumar – Portfolio

Personal portfolio website built with pure HTML, CSS, and JavaScript. No frameworks or dependencies required.

## 🚀 Deploy to GitHub Pages (3 steps)

1. **Create a new GitHub repository** named `portfolio` (or anything you like)

2. **Upload the file**
   - Go to your repo → click `Add file` → `Upload files`
   - Upload `index.html`
   - Commit with message: `Add portfolio`

3. **Enable GitHub Pages**
   - Go to repo `Settings` → `Pages` (left sidebar)
   - Under **Source**, select `Deploy from a branch`
   - Branch: `main` · Folder: `/ (root)`
   - Click **Save**

Your portfolio will be live at:
```
https://<your-github-username>.github.io/portfolio/
```

> It may take 1–2 minutes to go live after enabling Pages.


## ✏️ Customization

Open `index.html` in any text editor and update:

| What to change | Where to find it |
|---|---|
| LinkedIn / GitHub URLs | Hero section `<a href="...">` links |
| Email / phone | `contact-chip` spans and footer |
| Project descriptions | `proj-card` sections (01–04) |
| Skills | `skill-card` sections |

## 🛠 Tech Used

- HTML5 · CSS3 · Vanilla JavaScript
- Google Fonts (Syne, DM Mono, DM Sans) — loaded via CDN
- No build tools, no npm, no dependencies
